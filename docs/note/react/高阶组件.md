# 高阶组件

## 定义

**高阶组件是参数为组件，返回值为新组件的函数**

## 高阶函数

接收函数作为参数或将函数作为输出返回

常见的高阶函数有 `Array.prototype` 的 `map` `filter` `reduce` 等等

### 模拟一个map实现

```js
const map = (arr, fn) => {
  const newArr = [];
  
  for(item of arr) {
    newArr.push(fn(item));
  }
  
  return newArr;
}
const test = map([1,2,3], (a) => a * 2);
console.log(test); // [2,4,6]
```
