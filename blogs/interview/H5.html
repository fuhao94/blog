# 客户端通知前端

前端：windows.addEventListener()

客户端：windows.dispatchEvent()

// 直接自定义事件，使用 Event 构造函数：
var event = new Event('build');
// 监听事件
window.addEventListener('build', function (e) {}, false);
// 触发事件.
window.dispatchEvent(event);
